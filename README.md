# Surat House Finish

Premium home improvement website for **Surat House Finish** — a home improvement company in Surat, Gujarat.

## Tech Stack

- Next.js 15 (App Router)
- React 19
- TypeScript
- Tailwind CSS
- Lucide React Icons

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Project Structure

```
app/           # Next.js App Router pages & layouts
components/    # Reusable UI, layout, and section components
lib/           # Constants, SEO helpers, utilities
hooks/         # Custom React hooks
types/         # TypeScript type definitions
public/        # Static assets
styles/        # Global styles (app/globals.css)
```

## Configuration

Update business details in `lib/constants.ts`:

- Phone & WhatsApp number
- Email & address
- Site URL for SEO
